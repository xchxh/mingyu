# 提示词工坊

这是一个独立的提示词生成项目，当前主流程是：

1. 输入出生信息
2. 同时生成八字盘和紫微盘
3. 在结果页切换八字页、紫微页、提示词页
4. 在提示词页选择基于八字或紫微生成提示词
5. 直接复制，或调用系统分享把提示词正文发送到其他 AI 软件

它参考了两个现有项目的思路：

- `bz`：偏模板化的八字提示词拼装
- `zw`：偏结构化上下文的紫微提示词构建

结果页固定 3 个子页：

- 八字页
- 紫微页
- 提示词页

提示词页当前支持：

- 基于八字或紫微切换来源
- 选择年限
- 快捷问题
- 自定义问题
- 复制提示词
- 系统分享提示词正文

## 启动

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 当前边界

这个项目已经迁入核心排盘逻辑，但仍然聚焦“排盘后生成提示词”这条主线。

当前特点：

- 八字使用原项目核心排盘逻辑，支持真太阳时
- 紫微使用原项目核心排盘与分析载荷构建逻辑
- 提示词页面不再区分 `system prompt` / `user prompt`
- 分享是分享文本，不是分享链接
- 不直接调用大模型接口

如果后续你希望继续做，我建议下一步补这两类能力：

- 接回 AI 接口，做到“排盘后直接发送”
- 继续优化结果页的专业盘展示和移动端体验
